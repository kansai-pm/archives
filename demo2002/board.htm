関西オープンソース＋フリーウェア2002
デコレーションの作り方

文責：川合孝典(2002/12/10)

2002年12月6、7日におこなった「関西オープンソース＋フリーウェア2002」で表示したデコレーションの作り方です。

どんなものかは以下の写真の通り。当日、開始前のブースの写真です。

左 正面 右

　

手順

デコレーションとしてはラクダ掲示板、モモンガ、Tシャツ、大きなトラネコ、社名板の５つです。

社名板

まぁ見ての通りといった以上のことは、まったくないのですが。

  1. Windows付属のペイントを使って「Kansai Perl Mongers」をImpactというフォントで作成
    紙を有効利用するためA4の大きさで2行に分けて出力（Kansai PerlとMongers)
  2. それを印刷し、カッターで分割
  3. さらにトラネコと印鑑の画像を大きさを合わせて出力し、文字と一緒に１つに貼りあわせる。

モモンガ

プリンタドライバの「ポスタ作成」だけがポイントでしょう。

  1. モモンガの画像をプリンタで印刷
    その際、「ポスター印刷」で3x3を設定します（下図参照）。
    この辺りはプリンタに依存するでしょう。ちなみに私はPixus 850iを使用しています。
  2. 出力されたA4x9を貼りあわせ、さらに台紙となるプラスティックダンボールにはりつけます。
    今回は「のりしろ」と印刷された部分が透けてしまうため、裏に貼りました。本来はそうした文字はなくすか、ホワイトでつぶしてから貼りあわせ、表にはったほうが見やすいでしょう。

動的デモのスクリプト

ではスクリプトの内容について解説。（というほどのものでもありませんが...）

全体としてPerl/Tkで画像表示を、Win32::Soundで音楽の演奏を行っています。

Perl/Tkについての詳細は下記参考資料等をご覧ください。

$oMw->title('DEMO');
$oMw->FullScreen(1);    #これで全画面表示
my $oCvs = $oMw->Canvas(
                -background  => '#402500',
                -width       => $oMw->width,
                -height      => $oMw->height,
                -relief      => 'raised',
                -borderwidth => 0,
                -cursor =>'trek',
        )->grid;
dispImg();
MainLoop;
#--------------------------------------------------------------------
# 画像ファイルの表示
#--------------------------------------------------------------------
sub dispImg() {
    $oCvs->delete($oImg) if($oImg); #キャンバスからImageを削除
    $iPos %=scalar(@aImg);          #表示する画像を計算

    if(! defined($aoImg[$iPos])) {  
        my $oWk = $oMw->Photo(-file => $aImg[$iPos]->{img});
        $aoImg[$iPos]= $oWk;    
            #PhotoメソッドでImageを用意、配列に入れておけば一度きり
    }
    $oImg = $oCvs->createImage(     #キャンバスにImageを作成
                ($oMw->width)/2,
                ($oMw->height)/2,
                -image => $aoImg[$iPos]);
    if($iPos == 0) {                #$iPos==0つまり先頭にきたら
        $iSnd %= scalar(scalar(@aSnd));
        if($aSnd[$iSnd] ne '') {
            Win32::Sound::Play($aSnd[$iSnd], SND_ASYNC|SND_LOOP);
                    #指定されているファイルを演奏開始（非同期、繰り返し）
        }
        else {
            Win32::Sound::Stop; #ファイル名がなければ演奏停止
        }
        ++$iSnd;
    }
    $oMw->after($aImg[$iPos]->{wait}||5000, \&dispImg);
                                #指定された時間（デフォルトは5秒）後に
                                #この関数dispImgを呼び出し
    ++$iPos;
}

おまけ：GIFアニメを作成するスクリプト

ついでに上記のデモをGIFアニメ化するためのスクリプト。

Image::Magickを利用すれば、特に難しいところもありません。

唯一気をつけなければならないことは今回のJPEGファイルは大きなものだったので、先に小さく変換してからでないとメモリを圧迫してしまうということぐらいでしょうか。

use strict;
use Image::Magick;
my @aImg = (
    { img => 'KOF/スライド1.jpg',   },
    { img => 'KOF/スライド2.jpg',  },
    { img => 'KOF/スライド3.jpg',  },
    { img => 'KOF/スライド4.jpg',  },
    { img => 'KOF/スライド5.jpg',  },
    { img => 'KOF/スライド6.jpg',  },
    { img => 'KOF/スライド7.jpg',  wait  => 2000, },
    { img => 'KOF/スライド8.jpg',  wait  => 2000,  },
    { img => 'KOF/スライド9.jpg',  wait  => 1000,  },
    { img => 'KOF/スライド10.jpg', wait  => 3000,  },
    { img => 'KOF/スライド11.jpg', },
    { img => 'KOF/スライド12.jpg', },
    { img => 'KOF/スライド13.jpg', },
    { img => 'KOF/スライド14.jpg', },
    { img => 'KOF/スライド15.jpg', wait  => 1000, },
    { img => 'KOF/スライド16.jpg', wait  => 1000,},
    { img => 'KOF/スライド17.jpg', wait  => 1000,},
    { img => 'KOF/スライド18.jpg', },
    { img => 'KOF/スライド19.jpg', wait  => 6000},
    { img => 'KOF/スライド20.jpg', },
    { img => 'KOF/スライド21.jpg', },
    { img => 'KOF/スライド22.jpg', wait  => 2000 },
    { img => 'KOF/スライド23.jpg', },
    { img => 'KOF/スライド24.jpg', },
    { img => 'KOF/スライド25.jpg', wait  => 7000, },
    { img => 'KOF/スライド26.jpg', wait  => 3000, },
);
#1.画像を幅:200、高さ:150に縮小し、gifで出力
my $rhImg;
my $i=0;
foreach $rhImg (@aImg) {
    my $oImg = Image::Magick->new;
    $oImg->Read($rhImg->{img});
    $oImg->Resize('200x150');
    $oImg->Write(sprintf('KOF/slide%02d.gif', ++$i));
}
#2.そのファイルをまとめてGIFアニメ化
my $iDly;
my $oImg = Image::Magick->new;
$i=0;
#2.1 loopで繰り返しを設定
$oImg->Set(loop =>-1);
foreach my $rhImg (@aImg) {
    $oImg->Read(sprintf('KOF/slide%02d.gif', ++$i));
    $iDly = $rhImg->{wait};
    $iDly||=3000;   #本物よりも時間をちょっと短めに設定
    $iDly/=10;      #GIFアニメでは1/100秒単位
    #2.2　delayで移り変わるタイミングを調整
    $oImg->[$i-1]->Set(delay=>$iDly);
}
$oImg->Write('demo.gif');
