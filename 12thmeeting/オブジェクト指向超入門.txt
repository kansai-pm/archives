敷居を下げたい
プログラムとは、場合分けと繰り返しである
オブジェクト指向とは、場合分けをうまくやる方法である
…一つの試みとして



【例題〜CSVのカラム入れ替え】

my @order = split /,/, shift;       # 「3,1,2」のように順序を指定
while(<>){                          # 一行ずつ読んで
	chomp;                          # 改行は取る
	my @data = split /,/;           # 分解
	print join(',', map {$data[$_ - 1]} @order),"\n";  
                                    # 指定順序に組み立てて出力
}
